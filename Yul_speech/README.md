# 🦎 소리튠영어 My dream speech training

어린이와 학생들이 영어 스피치(발표)를 스스로 듣고 말하며 재밌게 발음을 연습할 수 있도록 돕는 인터랙티브 웹 애플리케이션입니다.

---

## 🌟 주요 기능 (Key Features)

1. **한 문장씩 연습 (Sentence Mode)**
   - 문장 단위의 집중 학습이 가능합니다.
   - 진행 상황을 한눈에 볼 수 있는 상단 진행 바(Progress Bar)를 지원합니다.
2. **원어민 발음 듣기 (TTS)**
   - `SpeechSynthesis` API를 활용하여 자연스러운 원어민 영어 오디오를 제공합니다.
   - 🐢 **목소리 속도 조절(0.6x ~ 1.3x)** 슬라이더를 통해 자신의 수준에 맞게 속도를 조절할 수 있습니다.
3. **내 목소리 녹음 및 발음 피드백 (STT)**
   - 구글 크롬 브라우저의 Web Speech API를 활용해 마이크로 녹음한 음성을 텍스트로 인식합니다.
   - 목표 문장과의 **발음 일치율(Similarity)**을 계산하여 3단계 맞춤 피드백 및 응원을 선사합니다.
4. **한글 번역 보기 토글**
   - 💡 필요에 따라 한글 번역 노출 여부를 토글하여 자기주도적 학습 효과를 높입니다.
5. **귀여운 마스코트 게코의 응원**
   - 각 단계마다 크레스티드 게코 캐릭터가 재미있고 신나는 격려 메시지를 보냅니다.

---

## 🚀 실행 방법 (How to Run)

본 애플리케이션은 별도의 서버 설치 없이 웹 브라우저 상에서 바로 실행되는 정적 SPA(Single Page Application)입니다. 단, **음성 인식(STT) 기능은 브라우저 보안 정책상 반드시 로컬 서버 환경 또는 HTTPS 환경에서 실행**되어야 정상 작동합니다.

### 추천 실행 방법 (로컬 웹 서버 사용)

#### 1. Python 활용 (간단한 방법)
터미널을 열고 프로젝트 폴더 경로에서 다음 명령을 실행합니다.
```bash
# Python 3.x 버전
python -m http-server 8000
```
그 후 브라우저에서 `http://localhost:8000` 주소로 접속합니다.

#### 2. Node.js `http-server` 활용
```bash
# http-server 설치 및 실행
npx http-server -p 8000
```
그 후 브라우저에서 `http://localhost:8000` 주소로 접속합니다.

## 🔒 브라우저 호환성 및 마이크 권한 안내

- **권장 브라우저**: **구글 크롬(Google Chrome)**
  - Web Speech API (음성 인식)는 구글 크롬에서 가장 완벽하게 작동합니다. Safari 등 타 브라우저에서는 마이크 인식이 원활하지 않을 수 있습니다.
- **보안 제약 사항**:
  - 브라우저 보안 규정에 따라 **`localhost`가 아니거나 `https://` 보안 프로토콜을 사용하지 않는 웹 주소**에서는 마이크 사용 권한 획득이 제한될 수 있습니다.
  - 마이크가 작동하지 않을 때는 브라우저 주소창 왼쪽의 **자물쇠 아이콘(🔒 또는 설정 아이콘)**을 클릭하여 **마이크 권한이 '허용'**되어 있는지 확인해 주세요.
